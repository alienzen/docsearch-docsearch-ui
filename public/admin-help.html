Sources

Le panneau « Sources » (et ses variantes SQL / web) permet d'enregistrer, d'activer/désactiver la recherche, de modifier le libellé et la description, et de retirer une source. Le nom d'une source (utilisé en interne, dans le registre et dans les documents déjà indexés) ne peut plus être modifié une fois créé — seuls le libellé affiché et la description peuvent être changés à tout moment.
